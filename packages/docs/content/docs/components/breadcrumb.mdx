## API Reference

{/* No exported Props interface — Breadcrumb sub-components use inline React.ComponentPropsWithoutRef types */}

All Breadcrumb sub-components extend their native HTML element props. `Breadcrumb` accepts a `separator` prop for custom separator rendering. `BreadcrumbLink` accepts an `asChild` prop to render a custom element via Radix `Slot`.
