## API Reference

{/* No exported Props interface — Alert uses inline VariantProps<typeof alertVariants> & React.HTMLAttributes<HTMLDivElement> */}

The `Alert` component accepts a `variant` prop (`"info"` | `"success"` | `"warning"` | `"critical"` | `"destructive"` | `"ai"` | `"neutral"`) plus all standard HTML `div` attributes. Sub-components `AlertIcon`, `AlertContent`, `AlertTitle`, and `AlertDescription` extend native HTML element props.
