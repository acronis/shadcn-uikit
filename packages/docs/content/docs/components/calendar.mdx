## API Reference

{/* No exported Props interface — Calendar extends React.ComponentProps<typeof DayPicker> with an additional buttonVariant prop */}

The `Calendar` component extends all `react-day-picker` `DayPicker` props and adds a `buttonVariant` prop that accepts any `Button` variant value to style navigation buttons.
