## API Reference

{/* No exported Props interface — DataTable uses generic DataTableProps<TData, TValue> internally */}

The `DataTable` component accepts `columns` (a `ColumnDef<TData, TValue>[]` array) and `data` (a `TData[]` array) as its primary props. It is built on `@tanstack/react-table` and provides sorting, filtering, pagination, and row selection out of the box.
