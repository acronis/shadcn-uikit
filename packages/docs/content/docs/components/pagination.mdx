## API Reference

{/* No exported Props interface — PaginationLinkProps is defined internally and not exported */}

Pagination sub-components extend their underlying HTML element props. `PaginationLink` accepts `isActive` (boolean) and `size` (from `ButtonProps`). `Pagination`, `PaginationContent`, `PaginationItem`, `PaginationPrevious`, `PaginationNext`, and `PaginationEllipsis` accept standard HTML element attributes.
